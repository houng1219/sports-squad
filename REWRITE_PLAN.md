# SportsSquad 改寫計畫

> 目標：把首頁從「冷冽夜空藍 SaaS 風」改造成「毛小愛式暖色親切風」  
> 改寫範圍：首頁（src/app/page.tsx）+ 全站配色（src/app/globals.css）+ Layout/NavBar 微調  
> 不動：API、Supabase、揪團 CRUD 頁、Profile 頁

---

## 1. 視覺規範

### 1.1 配色（從冷色 → 暖色）

| 用途 | 舊（冷色） | 新（暖色・毛小愛） |
|------|------------|---------------------|
| 背景 | `#0b1220` 深空藍 | `#FFF8F0` 米白 |
| 卡片底 | `#131c2e` | `#FFFFFF` |
| 主文字 | `#e8eef5` | `#3D2E1F` 深咖啡 |
| 次文字 | `#8a99b1` | `#8B7355` 暖灰 |
| 邊框 | `#1f2a3e` | `#F5E6D3` 米色 |
| **主色（強調）** | `#38bdf8` sky blue | `#FF7B54` 暖橘紅 |
| 主色 hover | `#0ea5e9` | `#E85A2A` |
| 副色 | `#5eead4` 薄荷 | `#FFB347` 鵝黃橘 |
| 強調底色 | `#0f2030` | `#FFE5D4` 淡橘底 |
| 成功 | `#34d399` | `#7DB46C` 暖綠 |
| 警示 | `#fbbf24` | `#F4A261` |

### 1.2 字體

- 中文：思源黑體（Noto Sans TC）→ **改用「Noto Sans TC Rounded」圓體或 LXGW WenKai 圓體**
- 英文：Geist Sans（保留）
- 標題：font-black + tracking-tight
- 內文：font-normal

### 1.3 形狀與質感

- 圓角：從 `rounded-2xl` (16px) → **rounded-3xl** (24px)
- 卡片陰影：從冷藍調 → 暖橘淡陰影（`rgba(255, 123, 84, 0.08)`）
- icon 容器：圓角方塊底色 → 改為圓形 + 暖色底

### 1.4 Emoji / 插圖策略

- Hero 右側加一隻 🏀🏸🏃 大三元組合插圖（純 CSS / emoji 排版）
- 卡片用 emoji 取代 lucide icon
- 球類 chip 用圓形 emoji 而不是方框

## 2. 新首頁結構（11 個區塊）

```
1.  Hero（暖橘漸層 + 毛孩式溫暖）
2.  三大數字（運動夥伴 / 覆蓋縣市 / 本月揪團）
3.  「什麼情境適合揪團？」情境分流（毛小愛招牌邏輯）
4.  探索球類（emoji 圓形 chip）
5.  最新揪團（暖色卡片）
6.  3 分鐘發起揪團（流程圖解）
7.  為什麼選 SportsSquad（信任三件套）
8.  真實揪團故事（見證牆）
9.  Q&A 常見問題（accordion，6 個）
10. 社會影響力（用揪團讓世界更好）
11. 最終 CTA + Footer
```

### 2.1 Hero（毛小愛式）

```
H1: 找到對的運動夥伴 一起流的汗最好玩
副標: 不用再揪不到人、不用再換球場、不用再尬聊陌生人
CTA 主: 🔥 3 分鐘發起揪團
CTA 次: 瀏覽附近揪團 →
視覺: 暖橘→鵝黃漸層 + 三大球 emoji 群（🏀🏸🏃）+ 浮動
```

### 2.2 情境分流（毛小愛招牌）

毛小愛用「貓咪 / 多毛孩家庭 / 不關籠」分流。我們用運動情境分流：

| 情境卡片 | 推什麼服務 |
|---------|------------|
| 🆕 剛到新城市、想找人一起動 | 智能推薦 |
| 🏃 一個人運動太無聊 | 瀏覽揪團列表 |
| 👥 想揪朋友但湊不滿人數 | 發起揪團 |
| 🎯 想固定時段、固定球友 | 加入 VIP 球隊（未來） |

每張卡片：icon + 情境 + 一句痛點 + 對應服務 CTA

### 2.3 3 分鐘流程

```
1️⃣ 選運動 + 地點 + 時間（30 秒）
2️⃣ 設定人數與費用（30 秒）
3️⃣ 一鍵發布，AI 推薦球友（10 秒）
✅ 揪團上線，開始有人報名
```

### 2.4 信任三件套

- **實名認證 + 評價系統**：每個揪團者都有公開評分
- **付款保障**：費用透過平台代收，活動完成才撥款
- **保險支援**：合作運動險，意外有理賠

### 2.5 見證牆

3 則真實揪團故事（從 mock-data 擴充）：
- 「週三羽球揪」 - 阿偉（每週穩定揪到 8 人）
- 「週末籃球三打三」 - Rex（從 3 人湊到 12 人）
- 「路跑新手團」 - Andy（陪 5 個跑友完成半馬）

### 2.6 Q&A（6 個 accordion）

1. 揪團要付費嗎？
2. 怎麼報名揪團？需要先註冊嗎？
3. 找不到適合的球類怎麼辦？
4. 揪到的球友可以信任嗎？
5. 如果活動取消，報名費會退嗎？
6. 怎麼發起我的第一個揪團？

### 2.7 社會影響力

- **用揪團解決孤單運動**：統計 73% 的運動愛好者覺得「找不到人一起動」
- **疫情後的健康社交**：揪團讓 5,000+ 人重新開始運動
- **女性友善揪團**：過濾機制、評價機制保障女性球友

## 6. 風險 / 取捨

- **Rex 的偏好**：原本是酷冷 SaaS 風格，改暖色等於**品牌定位大改**。如果未來想做 B2B 企業客戶或男性重度運動社群，暖色可能不對味。
- **替代方案**：保留深色作為 B2B 主題，淺色作為 C2C 主題（用路徑切換）。但 MVP 階段先統一暖色。
- **字體大小**：Noto Sans TC Rounded 在 14px 以下的渲染，建議字級最小 15px。
